# Changelog

本项目的用户可见变更记录于此。版本号遵循 [Semantic Versioning](https://semver.org/)。

## [0.1.0] - 2026-07-16

首个公开版本。

### 新增

- Windows x64 桌面悬浮窗和系统托盘菜单。
- 从 cc-switch SQLite 数据库读取 Claude Code Token 与费用数据。
- 樱桃消耗动画，以及当前用量、近 30 分钟速度和每日消费告警灯。
- 日、周、月和全部统计周期；支持全局及当前项目范围。
- 中文 / English 界面、点击穿透、透明度、缩放和布局配置。
- 手动 JSON 数据源与自定义模型价格。
